delete (   $url,
  $data = [],
  $format = 'json' 
)

Informa que a requisição deverá enviar um DELETE.

Parâmetros
string$urlCaminho que será realizada a requisição.
array$dataDados que serão enviados pela requisição. (Padrão: Sem dados)
string$formatInforma o formato que será retornado os dados. (Aceitos: json [padrão], xml e text)
Retorna
mixed Retorno em formato configurado.

execute (   $url,
  $data = [],
  $method = 'GET',
  $header = [] 
)

Executa uma requisição http com as opções de parametro informadas.

Parâmetros
string$urlCaminho que será realizada a requisição.
string$methodTipo de cabeçalho que será informado na requisição. (Padrão: GET)
array$dataDados que serão enviados pela requisição. (Padrão: Sem dados)
array$headerInformação de cabeçalho adicionais que serão utilizadas. (Padrão: Sem dados)
Retorna
string Retorno em formato string da requisição.

Implementado em cURL e Stream.
